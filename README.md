# EventSphere

A full-stack event ticket booking platform built using Spring Boot and React that allows users to browse events, purchase tickets, and manage bookings through a modern and responsive interface.

---

## Features

- User Registration and Login
- JWT Authentication
- Role-Based Access Control
- Event Creation and Management
- Ticket Booking System
- QR Code Validation
- Secure REST APIs
- Event Dashboard
- Responsive User Interface
- Event Search and Filtering

---

## Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- PostgreSQL
- Maven

### Frontend
- React
- TypeScript

## Backend Setup

```bash
cd backend
```

Configure your database inside:

```properties
application.properties
```

Run the application:

```bash
./mvnw spring-boot:run
```

Backend runs on:

```
http://localhost:8080
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
